**EPS (Esquilona Prime Stay)** is a modern **Renting House Management System** built with **Laravel** and the **TailAdmin Laravel Starter Kit**. This system helps property owners and administrators efficiently manage rental properties, tenants, and related transactions.

---

## Features

- **Admin Dashboard**  
  Intuitive dashboard to monitor properties, tenants, and rental activities.

- **Property Management**  
  Add, update, and remove houses or rental units easily.

- **Tenant Management**  
  Keep track of tenants, their rental contracts, and personal details.

- **Rental Transactions**  
  Manage monthly payments, track overdue payments, and generate reports.

- **Notifications**  
  Optional notifications for tenants regarding rent due or updates.

- **Authentication & Roles**  
  Secure login for Admins and optional tenant access with role-based permissions.
